# rakia-content

Pre-built content collection packs for [Rakia](https://github.com/rakia-app) — the Hebrew
Torah-library desktop app.

Each release asset `<slug>-v<version>.zip` is a self-contained pack (books + division
hierarchy + base↔commentary links + rendered HTML) that the app installs by "replay",
with no live fetch. `catalog.json` lists the available packs. Hosting here (not on
sefaria.org) keeps the texts reachable on filtered networks.
